What we're building

Every action an AI agent takes carries a signed proof of the policy it checked before execution. Shellproof is an infrastructure layer that wraps any autonomous agent in a verifiable compliance shell.\n\nAgents declare their constraints once. Shellproof monitors runtime behavior, generates zero-knowledge attestations, and logs them on-chain. If an agent violates its declared policy, the proof fails and the transaction is halted before settlement.
