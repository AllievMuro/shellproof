The Invisible Admin Problem

In 2023 and 2024, over $2.8B in on-chain assets were drained through exploits that began with a single privileged function call — an upgrade, a pause, a role reassignment — that nobody caught in time. The tragedy isn't just the loss. It's that the evidence existed on-chain the entire time, buried in raw calldata that no one was watching.

DAO governance documents say one thing. The actual execution record says another. When a solo admin wallet calls a multisig-guarded function at 3am UTC, there is no alert, no receipt, no structured forensic trail that an insurance underwriter or a governance committee can act on. Protocol teams are flying blind, patching after the fact, and issuing post-mortems with incomplete evidence.

Existing audit tools snapshot code at a single point in time. They don't watch runtime behavior. They don't prove who called what, when, and whether it was authorized. That gap between static audit and live execution is where sophisticated attackers operate — and where reputational and financial damage compounds.

Non-Invasive. Cryptographically Binding. Fully On-Chain.

Shellproof deploys as an ERC-2535 Diamond-compatible proxy that wraps your existing contract deployments without requiring redeployment or source modifications. It intercepts privileged function selectors — upgradeTo(), pause(), transferOwnership() — at the proxy layer before execution reaches your logic.

Each intercepted call triggers a Groth16 ZK proof generated via SnarkJS, cryptographically binding the caller address, function selector, block timestamp, and block hash into a tamper-evident permission receipt. These receipts are anchored to EIP-4844 blob storage — fully on-chain, immutable, and ~80% cheaper than calldata equivalents.

A Base-native dashboard powered by The Graph lets security leads query structured logs, diff admin activity against declared governance rules, and fire real-time webhook alerts when anomalies are detected. Exportable JSON compliance bundles give protocol teams defensible forensic records for insurance underwriters, governance disclosures, and post-incident reporting — the moment they're needed.
